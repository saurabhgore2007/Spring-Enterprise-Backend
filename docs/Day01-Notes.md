## Key Definitions

### Spring Framework
Spring Framework is a lightweight, open-source Java framework that simplifies enterprise application development by providing Inversion of Control (IoC), Dependency Injection (DI), and many other enterprise features.

## ApplicationContext
ApplicationContext is the advanced Spring IoC container that extends BeanFactory and provides enterprise features such as events, internationalization, resource loading, and automatic bean management.

### IoC
Inversion of Control is a design principle in which the control of object creation and lifecycle is transferred from the application code to a container or framework.

## Spring Container
The Spring Container is the core component of Spring that creates, configures, manages, and injects Spring Beans.

### Dependency Injection
Dependency Injection is a design pattern in which required dependencies are provided to an object instead of the object creating them itself.

### Bean
A Spring Bean is an object that is instantiated, configured, and managed by the Spring IoC Container.

## Key Takeaways

- Spring promotes loose coupling.  
- Constructor Injection is preferred.  
- Beans are managed by the Spring Container.  
- IoC transfers object creation to the container.
